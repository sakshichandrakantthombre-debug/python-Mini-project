# Student Marksheet Project

## Description
This is a Python mini project to calculate student marks, total, percentage, grade, and pass/fail result.

## Features
- Student Name
- Roll Number
- Subject Marks
- Total Marks
- Percentage
- Grade Calculation
- Pass/Fail Result

## Technologies Used
- Python

## How to Run
1. Open the project in Jupyter Notebook or VS Code.
2. Run the Python file.
3. Enter student details and marks.
